# PASTRY P2P 
Implemented Pastry Algorithm using Distributed Hash Table, as discussed in the class by Prof. Vinay J. Riberio

How to run:
1. do "make"
2. then run, ./pastry

(For multiple nodes, run ./pastry in different terminals)

Functions implemented:
1. port
2. create
3. join
4. lset
5. put
6. get
7. routetable
